# Automatic_Files_Distributing_Application
Python GUI Project Automatic_Files_Distributing_Application. Managing files is an easy task. So, today letâ€™s head over building a file manager. This is a pretty easy application. A simple desktop application for management in particular directory which is separating various extension file from source folder to specific folder based on the file extension using python tkinter.
